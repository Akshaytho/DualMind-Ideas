## The Mission

## HARD CONSTRAINTS (violate any = idea is DEAD)

### 1. Zero sales required
- Users/customers must COME TO HIM — organic pull, not cold outreach
- No enterprise sales, no cold-calling CTOs, no relationship-dependent deals
- Distribution: Product Hunt, GitHub stars, Twitter/X virality, SEO, word of mouth
- Think: how did Notion, Figma, or Vercel grow? Users found THEM

### 2. Solo-buildable with Claude AI
- One developer (Akshay) + Claude AI must be able to build and ship MVP in 4-6 weeks
- No team needed. No co-founder dependency. No hiring before revenue
- Tech stack Akshay already knows: React Native, Fastify, PostgreSQL, Python, Claude API

### 3. No licenses or regulatory barriers
- No FIU license, no RBI approval, no ₹2 crore net worth requirements
- No legal gray areas. Clean business from day one
- No dependency on government partnerships or institutional access

### 4. No existing competitors
- Search the web thoroughly. If ANYTHING similar exists, KILL IT
- "Nobody does it exactly like this" is NOT good enough — if the market exists, someone will copy
- Must be a genuinely unnoticed problem

### 5. Monopoly through data or network effects
- Using the product must generate data that makes it better (flywheel)
- OR: every new user must make it more valuable for existing users (network effect)
- Razorpay/Google/big tech should NOT be able to replicate it easily
- Ask: "What does Akshay have after 1 year that nobody can copy?"

### 6. Revenue from day one (or very fast)
- Freemium OK, but clear path to paid within first month
- Users must feel "I would pay for this" immediately
- No "grow first, monetize later" — that's a VC game, not a solo dev game
