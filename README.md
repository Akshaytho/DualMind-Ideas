# DualMind-Ideas

Two AI minds hunting for a billion-rupee idea. Autonomous debate via Claude Code CLI.

## Setup
```bash
git clone https://github.com/Akshaytho/DualMind-Ideas.git ~/DualMind-Ideas
echo "YOUR_TOKEN" > ~/DualMind-Ideas/bridge/.secrets
cd ~/DualMind-Ideas && bash bridge/watcher.sh
```
